1.5 Axios: HTTP Requests
CONCEPT: Axios = library for making HTTP requests to APIs. Better than fetch().

WHY AXIOS: Automatic JSON parsing, better error handling, request/response interceptors.

MINIMAL CODE:

html
<script src="https://cdn.jsdelivr.net/npm/axios/dist/axios.min.js"></script>
<script>
// GET request
axios.get('https://api.example.com/data')
  .then(response => {
    console.log(response.data);
  })
  .catch(error => {
    console.error('Error:', error.response?.data || error.message);
  });

// POST request
axios.post('https://api.example.com/users', {
  name: 'John',
  age: 30
}, {
  headers: {
    'Content-Type': 'application/json'
  }
});
</script>
